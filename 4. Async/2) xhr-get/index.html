Async/xhr-get

Задача:

Задача написать функцию, которая будет отправлять GET запрос по указаному URL (использовать XMLHttpRequest) Добавлять заголовок Content-Type: 'application/json' Функция должна возвращать promise Если запрос успешный (статус равен 2xx), то resolve с данными, которые вернул сервер (предположим что сервер будет возвращять JSON) Если не удачный, то вернуть ошибку от сервера
