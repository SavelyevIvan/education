Задача:

Написать функцию, которая будет сравнивать объекты

Пример

isEqual({}, {}) === true
isEqual({a: 1}, {a: { a: 1 }}) === false
isEqual({ a: { b: 1 }}, { a: { b: 1 }}) === true
isEqual({ a: { b: 1 }}, { a: { b: null }}) === false
isEqual({ a: null }, { a: null }) === true
isEqual({ a: undefined }, { a: null }) === false
isEqual({ a: undefined }, { a: undefined }) === true

Замечание:

Значение объекта может быть объектом
